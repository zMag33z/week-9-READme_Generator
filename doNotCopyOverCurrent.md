# **The Dev Tech Blog**

### A CMS-style blog site similar to a Wordpress site.  Following the MVC paradigm in its architectural structure, using Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication.  Allowing developers to publish their own blog posts and comment on other developersâ€™ posts as well.<br><br>

## Usage:
  
When first entering the site one will be able to view all posts and comments to those posts.  Once a user signsup or logs in they'll be able to add a comment to a post on the homepage or update/delete content they create on their personal dashboard.
